## 🧠 What I Learned

- CSS Reset (`* { margin: 0; padding: 0; box-sizing: border-box; }`)
- Flexbox for layout (navbar, columns, cards, footer)
- `background-size: cover` for hero images
- `rgba()` alpha transparency for subtle text
- Google Fonts integration & font pairing
- Semantic HTML (`header`, `main`, `section`, `footer`, `nav`, `form`)
- Reusable components (shared `.btn` class)
- Form accessibility (`label for` + `input id` linking)
- Hover transitions with `transition` property
